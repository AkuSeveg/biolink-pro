# 🚀 BioLink Builder Pro - White-Label SaaS
**100% Client-Side BioLink Generator | No Database Needed | Plug & Play**

Selamat datang di **BioLink Builder Pro**! Ini adalah *source code* aplikasi web (SaaS) premium yang memungkinkan pengguna Anda membuat halaman BioLink keren secara *real-time* tanpa perlu mendaftar atau menggunakan *database*. 

Aplikasi ini menggunakan teknologi 100% Vanilla JavaScript, sehingga Anda bisa melakukan *hosting* secara gratis di Vercel, GitHub Pages, atau Netlify. Sangat cocok untuk dijual kembali (White-Label) atau dijadikan layanan jasa pembuatan BioLink!

---

## ✨ Fitur Unggulan (Killer Features)

*   **📱 Live Split-Screen Builder:** Pengguna dapat mengetik data di panel kiri dan langsung melihat hasilnya di Mockup HP interaktif di sebelah kanan secara *real-time*.
*   **🎨 Multi-Theme System:** Dilengkapi dengan 3 tema premium bawaan:
    *   *Glassmorphism (Modern & Transparan)*
    *   *Dark Cyberpunk (Elegan & Gelap)*
    *   *Clean Minimalist (Cerah & Profesional)*
*   **✨ Premium Animations:** Dilengkapi dengan animasi *Typewriter* (Ketik-hapus otomatis pada teks Bio) dan *Scroll Entrance* yang membuat hasil BioLink terlihat sangat mewah.
*   **⬇️ 1-Click HTML Export:** Setelah selesai mendesain, cukup tekan tombol "Download", dan sistem akan merakit satu file `.html` mandiri yang siap di-*hosting* di mana saja oleh pengguna.
*   **⚡ Zero Server Cost:** Tidak butuh Node.js backend, tidak butuh database MySQL/MongoDB. Semua berjalan sangat cepat di *browser* pengunjung Anda.

## 🚀 Cara Menjalankan & Deploy

Aplikasi ini sangat mudah diatur. Karena bersifat *Client-Side*, Anda bahkan tidak perlu melakukan proses *build*.

### 1. Uji Coba Lokal (Di Komputer Anda)
1. Ekstrak *file source code* yang telah Anda unduh.
2. Cukup klik dua kali pada file `index.html` untuk membukanya di *browser* (Chrome/Edge/Safari).
3. Aplikasi Builder sudah siap digunakan!

### 2. Deploy ke Internet (Gratis menggunakan Vercel)
1. Buat akun gratis di [Vercel.com](https://vercel.com).
2. Di *dashboard* Vercel, klik **"Add New Project"**.
3. Pilih opsi folder project ini di perangkat Anda, lalu klik unggah / tarik ke halaman Vercel.
4. Tunggu beberapa detik, dan aplikasi BioLink Builder Anda sudah *Live* dan siap dibagikan ke seluruh dunia!
